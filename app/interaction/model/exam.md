**1. 证明实对称矩阵的特征值为实数。**

**证明：**
设 $A$ 为 $n$ 阶实对称矩阵，即满足 $A=A^T$ 且矩阵元素均为实数（$\bar{A}=A$）。
设 $\lambda$ 为 $A$ 的任一特征值，$\vec{x}$ 为对应的非零特征向量，则有：
$$A\vec{x}=\lambda\vec{x}$$
对上式两边取复数共轭，由于 $A$ 是实矩阵，得到：
$$A\bar{\vec{x}}=\bar{\lambda}\bar{\vec{x}}$$
对该式两边取转置，利用 $A=A^T$，得到：
$$\bar{\vec{x}}^TA=\bar{\lambda}\bar{\vec{x}}^T$$
将上式两边同时右乘向量 $\vec{x}$，得到：
$$\bar{\vec{x}}^TA\vec{x}=\bar{\lambda}\bar{\vec{x}}^T\vec{x}$$
另一方面，将初始方程 $A\vec{x}=\lambda\vec{x}$ 两边同时左乘 $\bar{\vec{x}}^T$，得到：
$$\bar{\vec{x}}^TA\vec{x}=\lambda\bar{\vec{x}}^T\vec{x}$$
对比上述两式，可得：
$$\bar{\lambda}\bar{\vec{x}}^T\vec{x}=\lambda\bar{\vec{x}}^T\vec{x}$$
因为 $\vec{x}$ 为非零特征向量，其内积 $\bar{\vec{x}}^T\vec{x}=\sum_{i=1}^n|x_i|^2>0$。
两边同时约去非零项 $\bar{\vec{x}}^T\vec{x}$，得到 $\bar{\lambda}=\lambda$。
故 $\lambda$ 必为实数，实对称矩阵的特征值均为实数得证。

**2. 如果把矩阵 $A\in\mathbb{R}^{2\times2}$ 看成是平面变换 $\vec{y}=A\vec{x}$，叙述矩阵 $A$ 的特征值和特征向量在该平面变换中的含义。**

**解答：**
在平面线性变换中：
* **特征向量**代表了变换的“不变主轴”。即空间中沿着特征向量方向的直线，在经过矩阵 $A$ 映射后依然是同一条直线，方向不发生偏转（最多只发生反向）。
* **特征值**代表了沿着对应特征向量方向上，空间被“拉伸或压缩的比例因子”。若特征值绝对值大于 $1$，表示该方向被拉伸；若介于 $0$ 和 $1$ 之间，表示被压缩；若为负数，则表示在该方向上还发生了关于原点的对称翻转。

**3. 用 $\varepsilon-\delta$ 语言叙述函数 $f(x)$ 在区间 $(a,b)$ 一致连续函数的定义。设 $f(x)$ 为 $(a,b)$ 上一致连续，证明 $f(x)$ 在 $(a,b)$ 上有界。**

**定义：**
函数 $f(x)$ 在区间 $(a,b)$ 上一致连续，是指：对于任意给定的 $\varepsilon>0$，都存在一个只与 $\varepsilon$ 有关的 $\delta>0$，使得对于区间 $(a,b)$ 内的任意两点 $x_1,x_2$，只要 $|x_1-x_2|<\delta$，就一定有 $|f(x_1)-f(x_2)|<\varepsilon$ 成立。

**证明：**
已知 $f(x)$ 在 $(a,b)$ 上一致连续。
取 $\varepsilon=1$，存在全局的 $\delta>0$，使得 $\forall x',x''\in(a,b)$，当 $|x'-x''|<\delta$ 时，有 $|f(x')-f(x'')|<1$。
取区间内一固定点 $x_0\in(a,b)$。对于任意 $x\in(a,b)$，线段 $[x_0,x]$ 的长度必定严格小于 $b-a$。
我们将从 $x_0$ 到 $x$ 的路径等分为 $N$ 段，取整数 $N=\lfloor\frac{b-a}{\delta}\rfloor+1$。
构造点序列 $x_0,x_1,\dots,x_k=x$（其中 $k\le N$），使得相邻两点间距 $|x_i-x_{i-1}|<\delta$。
根据绝对值三角不等式：
$$|f(x)-f(x_0)|\le\sum_{i=1}^k|f(x_i)-f(x_{i-1})|<k\cdot1\le N$$
由此可得，$|f(x)|<|f(x_0)|+N$。
由于 $x_0$ 是固定点，$|f(x_0)|$ 为常数，且 $N$ 是只依赖于区间长度和 $\delta$ 的有限常数。
因此，存在 $M=|f(x_0)|+N$，对所有 $x\in(a,b)$ 都有 $|f(x)|<M$，即证明了 $f(x)$ 在 $(a,b)$ 上有界。

**4. 叙述光学中的折射定律，并根据“两点之间光线传播路线使时间达到极小”这一原理证明折射定律。**

**定律叙述：**
光从光速为 $v_1$ 的介质 1 射入光速为 $v_2$ 的介质 2 时，入射角 $\theta_1$ 与折射角 $\theta_2$ 的正弦值之比等于光速之比：$\frac{\sin\theta_1}{\sin\theta_2}=\frac{v_1}{v_2}$。

**证明：**
设两介质交界面为 $x$ 轴。光线从点 $A(0,h_1)$ 出发，经过界面上的一点 $P(x,0)$，到达点 $B(L,-h_2)$。
光线传播的总时间 $t(x)$ 为：
$$t(x)=\frac{\sqrt{x^2+h_1^2}}{v_1}+\frac{\sqrt{(L-x)^2+h_2^2}}{v_2}$$
根据费马原理，光线传播时间达到极小，故对 $t(x)$ 求导并令其为 $0$：
$$\frac{dt}{dx}=\frac{1}{v_1}\frac{x}{\sqrt{x^2+h_1^2}}-\frac{1}{v_2}\frac{L-x}{\sqrt{(L-x)^2+h_2^2}}=0$$
由几何关系可知：$\sin\theta_1=\frac{x}{\sqrt{x^2+h_1^2}}$，$\sin\theta_2=\frac{L-x}{\sqrt{(L-x)^2+h_2^2}}$。
代入导数方程即得 $\frac{\sin\theta_1}{v_1}-\frac{\sin\theta_2}{v_2}=0$，移项后得到 $\frac{\sin\theta_1}{\sin\theta_2}=\frac{v_1}{v_2}$。折射定律得证。

**5. 写出随机变量和分布函数的定义。**

**解答：**
* **随机变量**：设随机试验的样本空间为 $\Omega$。如果对于 $\Omega$ 中的每一个样本点 $\omega$，都有一个确定的实数 $X(\omega)$ 与之对应，且对于任意实数 $x$，集合 $\{\omega:X(\omega)\le x\}$ 均构成一个事件，则称定义在 $\Omega$ 上的实值函数 $X(\omega)$ 为一个随机变量。
* **分布函数**：设 $X$ 是一个随机变量，$x$ 为任意实数，则函数 $F(x)=P(X\le x)$ 称为随机变量 $X$ 的累积分布函数，它刻画了随机变量取值落在区间 $(-\infty,x]$ 内的概率。

**6. 用 C 语言或者 Matlab 编写算法程序，求解以下关于函数 $y(x)$ 的二阶微分方程边值问题：**
$$\begin{cases}y''+xy'+5x=0,\\ y(0)=0,y(1)=1\end{cases} \quad (0<x<1)$$

**程序设计思路：**
本题采用**有限差分法（Finite Difference Method）**进行数值求解。
1. **网格剖分**：将区间 $[0,1]$ 进行 $N$ 等分，步长 $h=1/N$，离散节点记为 $x_i=i\cdot h$。
2. **离散化导数**：利用中心差分格式逼近导数：$y''\approx\frac{y_{i+1}-2y_i+y_{i-1}}{h^2}$，$y'\approx\frac{y_{i+1}-y_{i-1}}{2h}$。
3. **构建线性系统**：将差分格式代入原微分方程，整理得到关于未知节点 $y_i$ 的三对角线性方程组 $A\mathbf{y}=\mathbf{b}$。
4. **边界处理**：将边界条件 $y_0=0$ 和 $y_N=1$ 代入方程组首尾，利用 MATLAB 的矩阵左除 `\` 运算高效求出内点数值解，并拼接绘制曲线。

**MATLAB 源代码：**
```matlab
% 求解 y'' + xy' + 5x = 0, 边界条件: y(0)=0, y(1)=1
N = 100; % 网格数量
h = 1 / N; % 步长
x = linspace(0, 1, N+1)'; % 离散节点

% 构造三对角矩阵 A 和右端向量 b (针对内部节点 i = 1 to N-1)
A = zeros(N-1, N-1);
b = zeros(N-1, 1);

for i = 1:(N-1)
    xi = x(i+1); % MATLAB 索引从 1 开始，x(i+1) 对应真实的第 i 个内部节点
    
    A(i, i) = -2 / h^2;
    if i > 1
        A(i, i-1) = 1/h^2 - xi/(2*h);
    end
    if i < N-1
        A(i, i+1) = 1/h^2 + xi/(2*h);
    end
    b(i) = -5 * xi;
end

% 边界条件处理
% y(0) = 0, 对 b(1) 无影响
% y(1) = 1, 将其移项至最后一个内点方程的右侧
b(N-1) = b(N-1) - (1/h^2 + x(N)/(2*h)) * 1;

% 求解三对角线性方程组
y_interior = A \ b;

% 拼接首尾边界值
y = [0; y_interior; 1];

% 绘制函数曲线
figure;
plot(x, y, 'b-', 'LineWidth', 2);
xlabel('x'); ylabel('y(x)');
title('y^{\prime\prime} + xy^\prime + 5x = 0 边值问题数值解');
grid on;
